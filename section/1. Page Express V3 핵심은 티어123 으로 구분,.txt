1. Page Express V3 핵심은 티어1/2/3 으로 구분, 티어1은 공통 기능 및 백엔드-프론트엔드 연결 과정에서 필요한 디자인-기능인데, 대부분은 정상이지만, 개발 과정에서 발생한 비효율적인 코드를 최적화할거야

- 불필요한 주석 제거 (그룹 단위 제목은 허용)
- 반복 사용되는 코드는 재사용 가능하도록 모듈 또는 클래스로 묶어서 재활용
- js에 코딩된 스타일 속성은 css에 추가 (단, 불가피한 경우는 예외)
- 티어2 에서 공통적으로 사용하는 코드는 티어1 통합
- 코드는 용도에 맞게 그룹 단위 분류

2. Canvas 위자드 기능이 일부만 티어1에 묶였는데, 배경 이미지, 슬라이드, 이펙트 외 공통적으로 포함되는 텍스트 타이틀(h1), 보조 타이틀, 아이콘 버튼, 인디케이터 등 모든 요소가 Canvas 위자드 영역이며, 티어1에 포함되어야 함

- Page Express는 header 에서 사용하도록 page-v3 에서 적절하게 후보정
- Canvas Express는 Lite 모드는 풀스크린, Normal 모드는 가로 모드에는 좌측에 잘 보이도록 canvas-v3 에서 적절하게 후보정

* 현재 Canvas Express는 Canvas 위자드가 제대로 동작하지 않음, 특히 Contact 버튼 클릭 시 위자드 모드가 동작하면서, Form 이 자연스럽게 보여야 되는데, 새창으로 실행되는 문제가 있음, canvas-v1 코드를 참고해서 v3 규격에 맞게 문제 없이 동작하도록 개선
